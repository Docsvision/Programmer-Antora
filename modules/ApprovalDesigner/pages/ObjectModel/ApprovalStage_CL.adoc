== Синтаксис

[source,csharp]
----
public class ApprovalStage : BaseCard

== Свойства

[cols=",",options="header"]
|===
|Имя |Описание
|`xref:ObjectModel/ApprovalStage.AdditionalSettings_PR.adoc[AdditionalSettings]` |Возвращает дополнительные настройки этапа согласования.
|`xref:ObjectModel/ApprovalStage.Decisions_PR.adoc[Decisions]` |Возвращает варианты решений для завершения этапа согласования.
|`xref:ObjectModel/ApprovalStage.MainInfo_PR.adoc[MainInfo]` |Возвращает основную информацию этапа согласования.
|`xref:ObjectModel/ApprovalStage.SelectedApprovers_PR.adoc[SelectedApprovers]` |Возвращает согласующих этапа согласования.
|`xref:ObjectModel/ApprovalStage.States_PR.adoc[States]` |Возвращает возможные состояния этапа согласования.
|`xref:ObjectModel/ApprovalStage.TaskSettings_PR.adoc[TaskSettings]` |Возвращает настройки заданий участников этапа согласования.
|===
