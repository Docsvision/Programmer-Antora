Содержит системную информацию карточки (секция "Системные свойства").

* *Пространство имён:* `xref:Platform-ObjectModel:ObjectModel_NS.adoc[DocsVision.BackOffice.ObjectModel]`
* *Сборка:* `DocsVision.BackOffice.ObjectModel.dll`

== Синтаксис

[source,csharp]
----
public class BaseCardSystemInfo : BaseCardSectionRow
----

== Свойства

[cols=",",options="header"]
|===
|Имя |Описание
|`CardKind` |Задаёт или возвращает вид карточки.
|`State` |Задаёт или возвращает состояние карточки.
|===

[cols=",",options="header"]
|===
|Имя |Описание
