== Пространства имен

[cols=",",options="header",]
|===
|Пространство имен |Описание
|xref:ADSync/ADSync_NS.adoc[ADSync] |Содержит интерфейс и перечисление характеризующие возможность синхронизации базы Docsvision и домена основанного на ActiveDirectory.
|===
