StaffUnit GetDepartment(Guid unitId)

unitId:::
Тип: http://msdn.microsoft.com/ru-ru/library/system.guid.aspx[System.Guid]

Тип: xref:api/DocsVision/BackOffice/ObjectModel/StaffUnit_CL.adoc[StaffUnit]
