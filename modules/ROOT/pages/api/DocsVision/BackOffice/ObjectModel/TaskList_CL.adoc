== Свойства

[cols=",",options="header",]
|===
|Имя |Описание
|Processes |Возвращает коллекцию бизнес-процессов прикрепленных к карточке.
|SystemInfo |Возвращает вид и состояние карточки.
|TaskGroups |Возвращает коллекцию групп заданий.
|Tasks |Возвращает коллекцию заданий.
|===
