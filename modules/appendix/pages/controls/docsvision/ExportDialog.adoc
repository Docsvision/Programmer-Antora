= Элемент управления "ExportDialog"

Элемент управления `_ExportDialog_` предоставляет функциональность экспорта и печати карточки. Не имеет аналогов в _Конструкторе разметок_.

.Элемент управления "ExportDialog"
image::ROOT:export-dialog.png[Элемент управления "ExportDialog"]

Данный компонент не может быть использован в карточке как элемент управления, он должен вызываться по событию (см. описание класса `xref:BackOffice-WinForms:Controls/ExportForm_CL.adoc[DocsVision.BackOffice.WinForms.Controls.ExportForm]`).
